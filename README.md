# AI Agent Workflow Builder

시각적으로 AI Agent 워크플로우를 빌드하고 실행할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- 🎨 **드래그 & 드롭 워크플로우 에디터**: React Flow 기반의 직관적인 노드 기반 에디터
- 🤖 **다양한 노드 타입**: LLM, API, Function, Condition 등 다양한 노드 지원
- ⚡ **실시간 실행**: 워크플로우 실행 및 실시간 로그 스트리밍
- 💾 **워크플로우 저장/불러오기**: JSON 형식으로 워크플로우 저장 및 불러오기
- 🎯 **타입 안전성**: TypeScript로 구현된 완전한 타입 안전성

## 기술 스택

- **Frontend**: React 19 + TypeScript
- **빌드 도구**: Vite
- **상태 관리**: Zustand
- **워크플로우 에디터**: React Flow
- **아이콘**: Lucide React

```bash

```

## 사용 방법

1. **노드 추가**: 왼쪽 팔레트에서 원하는 노드를 드래그하여 캔버스에 드롭합니다.
2. **연결 생성**: 노드의 핸들을 드래그하여 다른 노드와 연결합니다.
3. **워크플로우 실행**: 상단 툴바의 "실행" 버튼을 클릭하여 워크플로우를 실행합니다.
4. **로그 확인**: 오른쪽 패널에서 실시간 실행 로그를 확인할 수 있습니다.
5. **저장/불러오기**: 툴바에서 워크플로우를 저장하거나 불러올 수 있습니다.

## 노드 타입

### Start Node
워크플로우의 시작점입니다.

### LLM Node
LLM API를 호출하는 노드입니다. OpenAI, Anthropic, Gemini 등 다양한 제공자를 지원합니다.

### API Node
REST API를 호출하는 노드입니다. GET, POST, PUT, DELETE 메서드를 지원합니다.

### Function Node
JavaScript 또는 Python 코드를 실행하는 노드입니다.

### Condition Node
조건 분기를 처리하는 노드입니다. True/False 경로를 설정할 수 있습니다.

### End Node
워크플로우의 종료점입니다.

## 프로젝트 구조

```
src/
├── components/          # React 컴포넌트
│   ├── WorkflowEditor/  # 워크플로우 에디터
│   ├── NodeTypes/       # 각종 노드 타입 컴포넌트
│   ├── NodePalette/     # 노드 팔레트
│   ├── Toolbar/         # 툴바
│   └── LogPanel/        # 로그 패널
├── stores/              # Zustand 상태 관리
├── types/               # TypeScript 타입 정의
├── engine/              # 워크플로우 실행 엔진
└── App.tsx              # 메인 앱 컴포넌트
```
